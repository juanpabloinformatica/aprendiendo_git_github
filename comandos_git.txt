git and github

-1 inside the folder -> git init
-2 check status -> git status -s
-3 first modificatin -> git add .
-4 second modification (box signature) -> git commit -m "message body"
-5 to look for boxes previously created ->  git log --oneline

if appears vim editor -> press :q!

if you want to check user config -> git config user.email or git config user.name
